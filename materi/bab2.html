2.3 Hyperlink, List, dan Gambar

Tiga elemen fundamental yang membuat web menjadi kaya informasi dan terhubung satu sama lain.

2.3.1 Hyperlink (Anchor)

Tag <a> adalah inti dari World Wide Web.

  • href: Tujuan link (URL Absolute, Relative, atau Anchor ID).
  • target="_blank": Membuka link di tab baru (wajib untuk link eksternal).

2.3.2 List (Daftar)

HTML menyediakan tiga jenis list untuk mengorganisir data:

  • Unordered List (<ul>): Daftar dengan simbol bullet (urutan tidak penting).
  • Ordered List (<ol>): Daftar dengan nomor/huruf (urutan penting).
  • Description List (<dl>): Daftar definisi istilah.

2.3.3 Gambar (<img>) dan Figure

Mempercantik web dengan konten visual.

  • Tag <img>: Bersifat self-closing. Wajib gunakan atribut alt untuk SEO & aksesibilitas.
  • Figure & Figcaption: Tag semantik untuk membungkus gambar dengan teks keterangannya.

2.4 Multimedia (Audio, Video)

HTML5 menghadirkan dukungan multimedia native tanpa memerlukan plugin eksternal seperti Flash.

2.4.1 Audio (<audio>)

Digunakan untuk memutar suara atau musik latar.

  • Format Support: MP3 (Universal), OGG, WAV.
  • Atribut: controls, autoplay, loop, muted.

2.4.2 Video (<video>)

Digunakan untuk memutar video.

  • Format Support: MP4 (H.264), WebM.
  • Atribut: poster (thumbnail sebelum diputar), width, height.

2.5 Form & Input Modern

2.5.1 Tipe Input Baru HTML5

Meningkatkan UX dengan keyboard yang sesuai di mobile.

Tipe Deskripsi Tampilan Mobile
email Validasi format '@' Keyboard dengan '@' dan '.'
number Hanya angka, atribut min/max/step Numpad (Angka Besar)
date Kalender interaktif (Date Picker) Roda tanggal / Kalender Native
tel Input nomor telepon Dialer Pad
url Input alamat website Keyboard dengan '.com'
color Pemilih warna visual Palet warna / Roda warna

2.6 Semantic HTML

Semantic HTML berarti menggunakan tag yang memiliki makna (bukan hanya <div>). Ini sangat krusial untuk SEO dan aksesibilitas.

2.6.1 Daftar Elemen Semantik Utama

<header>

Kepala halaman/artikel. Berisi logo, judul, navigasi.

<nav>

Wadah khusus untuk menu navigasi (link).

<section>

Mengelompokkan konten dengan tema sama (misal: Bab).

<article>

Konten independen yang berdiri sendiri (blog post).

<aside>

Konten pendamping/sampingan (sidebar, iklan).

<footer>

Kaki halaman (copyright, kontak, peta situs).
