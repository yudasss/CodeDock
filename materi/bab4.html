4.1 Evolusi JavaScript: dari ES5 ke ES6+

JavaScript telah berkembang jauh sejak diciptakan pada tahun 1995. Perubahan terbesar dalam sejarah bahasa ini terjadi pada tahun 2015 dengan dirilisnya ES6.

4.1.1 Mengapa Harus Migrasi ke ES6+?

Berdasarkan tinjauan kurikulum industri teknologi terkini (Juan & Budi, 2023), penguasaan fitur-fitur ES6 merupakan prasyarat mutlak sebelum seorang pengembang dapat memahami kerangka kerja modern.

1. Efisiensi Sintaksis: Penulisan kode lebih ringkas dan mudah dibaca.
2. Keamanan Eksekusi: Mengurangi potensi bug seperti kebocoran variabel.
3. Arsitektur Modular: Memfasilitasi pengembangan aplikasi skala besar.

4.2 Variabel (let, const) dan Scope

Salah satu kelemahan arsitektural paling fatal pada JavaScript versi awal adalah penggunaan kata kunci var. ES6 menyelesaikan masalah ini dengan let dan const.

4.2.1 Block Scope vs Function Scope

  • var (Function Scope)
    Variabel "bocor" keluar dari blok logika (if, for).
  • let & const (Block Scope)
    Variabel terisolasi aman di dalam kurung kurawal {}.

4.2.2 Pedoman Penggunaan (Best Practices)

  • Prioritaskan const: Gunakan default untuk data yang tidak berubah.
  • Gunakan let Terbatas: Hanya jika nilai variabel perlu berubah.
  • Hindari var: Praktik buruk dalam kode modern.
// --- PENDEKATAN LAMA (BERISIKO) --- if (true) { var zonaBahaya = "Saya bocor ke luar!"; } console.log(zonaBahaya); // Output: "Saya bocor ke luar!" // --- PENDEKATAN MODERN (AMAN) --- if (true) { let zonaAman = "Saya aman di dalam sini."; const zonaKonstan = "Saya juga aman."; } // console.log(zonaAman); // Error: zonaAman is not defined

4.3 Arrow Function, Template Literals, Destructuring

Fitur-fitur ini disebut Syntactic Sugarâ€”membuat kode lebih mudah dibaca dan diekspresikan.

4.3.1 Arrow Function (=>)

Penulisan fungsi yang lebih ringkas dan perilaku this yang leksikal.

// ES5 const hitungLuas = function(p, l) { return p * l; }; // ES6 const luasCepat = (p, l) => p * l;

4.3.2 Template Literals (Backticks)

Menyisipkan ekspresi JS langsung ke string dengan ${...}.

console.log(`User ${user} memiliki skor ${skor}`);

4.3.3 Destructuring Assignment

Mengekstrak data dari Array/Object ke variabel terpisah.

const { namaLengkap: nama, alamat: { kota } } = mahasiswa;

4.4 Array & Object Methods Modern

ES6 menyediakan metode fungsional (Higher-Order Functions) untuk manipulasi data secara deklaratif.

4.4.1 .map()

Transformasi setiap elemen array.

const hargaDiskon = hargaDasar.map(item => item * 0.9);

4.4.2 .filter()

Penyaringan data berdasarkan kondisi.

const lulus = nilaiUjian.filter(nilai => nilai >= 70);

4.4.3 .reduce()

Agregasi data menjadi satu nilai.

const total = keranjang.reduce((acc, curr) => acc + curr, 0);

4.4.4 Spread Operator (...)

Mengekspansi array/object.

const hobiBaru = ['Gaming', ...hobiLama];
// Full Example const numbers = [1, 2, 3]; const doubled = numbers.map(x => x * 2); console.log(doubled); // [2, 4, 6]

4.5 DOM Manipulation dengan Query Selector

DOM adalah antarmuka pemrograman untuk dokumen web.

4.5.1 Standarisasi Selektor Modern

  • document.querySelector(): Elemen pertama.
  • document.querySelectorAll(): Semua elemen (NodeList).
const judul = document.querySelector('h1'); judul.textContent = "Diubah via JS"; judul.classList.add('text-blue-600');
