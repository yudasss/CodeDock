5.1 Pendahuluan Integrasi Komponen

Dalam pengembangan antarmuka web modern, penggunaan Utility-First CSS seperti Tailwind CSS yang dipadukan dengan logika JavaScript (ES6+) memungkinkan pengembang untuk membangun komponen yang interaktif tanpa bergantung pada pustaka pihak ketiga yang berat (seperti jQuery).

"Pendekatan yang digunakan dalam bab ini selaras dengan temuan dalam penelitian yang membandingkan framework CSS, di mana Tailwind CSS terbukti memberikan fleksibilitas desain yang lebih tinggi dan kode yang lebih ringkas dibandingkan pendekatan tradisional." (Putra, Lestari, & Kristianti, 2025).

5.2 Membuat Navigasi Responsif (Responsive Navbar)

Navigasi adalah elemen krusial dalam User Experience (UX). Berdasarkan prinsip Mobile-First, navigasi harus dapat beradaptasi dari layar desktop yang lebar menjadi menu "hamburger" yang ringkas pada perangkat seluler.

5.2.1 Konsep Dasar

Navbar responsif bekerja dengan memanipulasi properti display CSS.

  • a. Layar Besar (md:flex) Menu ditampilkan secara horizontal.
  • b. Layar Kecil (hidden) Menu disembunyikan secara default dan digantikan tombol toggle.
  • c. Interaksi JS Event listener mendeteksi klik pada tombol toggle untuk menambah/menghapus kelas hidden.

5.3 Membuat Slider Gambar (Carousel)

Slider gambar atau carousel sering digunakan untuk menampilkan hero section atau galeri produk. Meskipun banyak library tersedia (seperti SwiperJS), membuatnya dari nol (Vanilla JS) memberikan performa yang lebih baik.

5.3.1 Algoritma Slider

Logika dasar slider adalah manipulasi indeks array:

  • a Simpan daftar gambar atau elemen slide.
  • b Tentukan variabel currentIndex (mulai dari 0).
  • c Tombol "Next" menambah currentIndex (loop ke 0 jika akhir).
  • d Tombol "Prev" mengurangi currentIndex.
  • e Update tampilan UI berdasarkan indeks aktif.

5.3.2 Implementasi Kode

HTML Structure:
<div class="relative w-full max-w-4xl mx-auto mt-10"> <!-- Container Slide --> <div class="overflow-hidden relative h-64 rounded-lg shadow-lg"> <div class="slide absolute inset-0 transition-opacity duration-700 opacity-100"> <img src="img1.jpg" class="w-full h-full object-cover"> </div> <div class="slide absolute inset-0 transition-opacity duration-700 opacity-0"> <img src="img2.jpg" class="w-full h-full object-cover"> </div> <div class="slide absolute inset-0 transition-opacity duration-700 opacity-0"> <img src="img3.jpg" class="w-full h-full object-cover"> </div> </div> <!-- Navigasi --> <button id="prevBtn" class="absolute left-0 top-1/2 transform -translate-y-1/2 bg-gray-800 text-white p-2 rounded-r">❮</button> <button id="nextBtn" class="absolute right-0 top-1/2 transform -translate-y-1/2 bg-gray-800 text-white p-2 rounded-l">❯</button> </div>
JavaScript Logic:
const slides = document.querySelectorAll('.slide'); const nextBtn = document.getElementById('nextBtn'); const prevBtn = document.getElementById('prevBtn'); let currentSlide = 0; function showSlide(index) { // Reset semua slide opacity 0 slides.forEach((slide) => { slide.classList.remove('opacity-100'); slide.classList.add('opacity-0'); }); // Slide aktif opacity 100 slides[index].classList.remove('opacity-0'); slides[index].classList.add('opacity-100'); } nextBtn.addEventListener('click', () => { currentSlide = (currentSlide + 1) % slides.length; showSlide(currentSlide); }); prevBtn.addEventListener('click', () => { currentSlide = (currentSlide - 1 + slides.length) % slides.length; showSlide(currentSlide); });

5.4 Membuat Modal/Popup Interaktif

Modal adalah elemen UI yang muncul di atas konten utama (overlay), memaksa pengguna untuk berinteraksi dengannya sebelum kembali ke konten utama. Ini sering digunakan untuk formulir login, konfirmasi, atau notifikasi.

5.4.1 Teknik Overlay CSS

Kunci pembuatan modal dengan Tailwind adalah penggunaan utilitas posisi:

  • fixed: Agar modal tetap di layar meski di-scroll.
  • inset-0: Agar memenuhi seluruh layar (top, right, bottom, left = 0).
  • z-50: Agar modal berada di lapisan paling atas.
  • bg-black bg-opacity-50: Membuat efek redup transparan di belakang modal.

5.4.2 Implementasi Kode

HTML & Tailwind:
<!-- Tombol Pemicu --> <button id="openModal" class="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"> Buka Modal </button> <!-- Struktur Modal (Hidden by default) --> <div id="modalOverlay" class="fixed inset-0 bg-gray-600 bg-opacity-50 hidden overflow-y-auto h-full w-full z-50"> <div class="relative top-20 mx-auto p-5 border w-96 shadow-lg rounded-md bg-white"> <div class="mt-3 text-center"> <h3 class="text-lg leading-6 font-medium text-gray-900">Konfirmasi</h3> <div class="mt-2 px-7 py-3"> <p class="text-sm text-gray-500"> Apakah Anda yakin ingin menyimpan data ini ke database? </p> </div> <div class="items-center px-4 py-3"> <button id="closeModal" class="px-4 py-2 bg-red-500 text-white text-base font-medium rounded-md w-full shadow-sm hover:bg-red-600 focus:outline-none focus:ring-2 focus:ring-red-300"> Tutup </button> </div> </div> </div> </div>
