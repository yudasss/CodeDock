1.1 Perkembangan Teknologi Web

Dunia pengembangan web (web development) telah mengalami evolusi yang sangat cepat dalam dua dekade terakhir. Secara historis, evolusi ini dapat dibagi menjadi tiga fase utama:

1.1.1 Web 1.0 (The Static Web)

Era di mana website hanya bersifat "baca saja" (read-only). Halaman web dibangun murni menggunakan HTML statis tanpa interaktivitas.

1.1.2 Web 2.0 (The Social Web)

Era interaktif saat ini. Pengguna bisa membuat konten (user-generated), berkomentar, dan berinteraksi. Teknologi AJAX dan JavaScript mendominasi.

1.1.3 Web 3.0 (The Semantic & Intelligent Web)

Masa depan web yang berfokus pada AI dan desentralisasi. Menuntut performa tinggi, aksesibilitas, dan struktur kode semantik.

1.2 Posisi Teknologi dalam Ekosistem Web

Frontend Development adalah praktik memproduksi HTML, CSS, dan JavaScript agar pengguna dapat melihat dan berinteraksi dengan website secara langsung.

1.2.1 HTML

(HyperText Markup Language): Struktur dasar konten.

1.2.2 CSS

(Cascading Style Sheets): Gaya visual, tata letak, dan responsivitas.

1.2.3 JavaScript

Interaktivitas dan logika pemrograman (client-side).

1.3 Alat Pengembangan

Pemilihan tools yang tepat sangat krusial untuk efisiensi dan produktivitas.

1.3.1 Visual Studio Code (Code Editor)

Standar industri editor kode. Ekstensi wajib: Live Server, Tailwind CSS IntelliSense, Prettier.

1.3.2 Browser & DevTools

Google Chrome/Firefox Developer Edition untuk debugging layout (Element Inspector), logika JS (Console), dan performa (Lighthouse).

1.3.3 Node.js dan NPM

Node.js untuk runtime JS di luar browser. NPM untuk mengelola paket/library seperti Tailwind CSS.

1.3.4 Git (Version Control)

Melacak perubahan kode, manajemen riwayat, dan kolaborasi tim.

1.4 Workflow Modern (Metode Waterfall)

1.4.1 Analisis

Identifikasi fitur dan kebutuhan halaman.

1.4.2 Perancangan

Desain UI dan struktur sistem.

1.4.3 Implementasi

Penulisan kode (Coding) HTML, CSS, JS.

1.4.4 Pengujian

Memastikan fitur berjalan sesuai spesifikasi.
